## 📋 撮影の手順

### ステップ1: サンプルデータの準備

1. **ログインする**
   - https://record-fishing-practice.pages.dev にアクセス
   - テストアカウントでログイン

2. **AREAタブでエリアを追加**
   - 北港エリア / スリット / 関門
   - 南港エリア / テトラ / 外向き
   - etc.

3. **BAITタブで餌を追加**
   - デフォルト餌（イガイ、ミジ貝等）を確認
   - カスタム餌を1-2個追加

4. **LOGタブで釣果を記録**
   - 通常の釣果（釣果数5、あたり数10等）
   - ボーズ（釣果数0、あたり数5等）
   - サイズ内訳あり
   - 餌内訳あり

### ステップ2: スクリーンショット撮影

#### 推奨ツール
- **PC**: Windows Snipping Tool / Mac Screenshot (Cmd+Shift+4)
- **スマホ**: 標準のスクリーンショット機能

#### 撮影設定
- **解像度**: 最低1280px以上
- **モード**: ダークモード（アプリのデフォルト）
- **ブラウザ**: Chrome / Safari（最新版）
- **ズーム**: 100%（拡大・縮小なし）

#### 撮影の順序（効率的な順番）

1. **LOGタブ**（No.1-9）
   - ログイン画面
   - AREA/LOCATION/POINT 選択
   - MAIN BAIT 選択
   - 釣果数・あたり数入力
   - あたり数エラー（意図的にエラーを発生させる）
   - BAIT BREAKDOWN
   - SIZE BREAKDOWN（詳細ON）
   - サイズ内訳エラー（意図的にエラーを発生させる）
   - CONDITION

2. **DATAタブ**（No.10-14）
   - フィルター
   - 月別推移グラフ
   - 円グラフ
   - ログ一覧（ボーズ・あたり表示）
   - CSV出力

3. **AREAタブ**（No.15-16）
   - AREA管理画面
   - エリア削除確認ダイアログ

4. **BAITタブ**（No.17）
   - BAIT管理画面

5. **編集・削除**（No.18-20）
   - 編集モーダル（DATAタブから）
   - 1件削除確認ダイアログ
   - 全削除確認ダイアログ（SETタブ）

6. **その他**（No.21-23）
   - iPhoneホーム画面追加
   - Androidホーム画面追加
   - バリデーションエラーまとめ

## 🎨 画像編集のポイント

### 推奨編集
1. **リサイズ**: 幅1280px程度にリサイズ（note.com での表示最適化）
2. **クロップ**: 不要な余白をトリミング（ただし、ヘッダーは残す）
3. **圧縮**: PNG圧縮（TinyPNG等）でファイルサイズを削減

### 避けるべき編集
1. **過度なトリミング**: タブ（LOG/DATA/AREA/BAIT/SET）は残す
2. **色の変更**: ダークモードの色を保持
3. **矢印・テキスト追加**: 説明文で補足するため、画像には追加しない

## 📝 note.com 投稿手順

### 1. 記事の作成

1. note.com にログイン
2. 「記事を書く」をクリック
3. タイトル: 「黒鯛釣果記録アプリ Ver.1.1.0 - 使い方ガイド」
4. 本文: USER_MANUAL_UPDATED.md の内容をコピー＆ペースト

### 2. 画像の挿入

1. 各セクションの説明の後に画像を挿入
2. 画像をドラッグ＆ドロップ or アップロード
3. 画像の説明文を追加（図X: ...）

### 3. 見出しの装飾

note.com では、以下のように見出しを装飾すると見栄えが良くなります:

```
## 🆕 5-6. あたり数（HIT COUNT）← NEW!
## 🆕 7. SIZE BREAKDOWN（サイズ内訳）← 拡張！
## 📊 9. DATA（分析）画面：見方← 拡張！
## 🗂️ 10. AREA（エリア管理）← NEW!
## 🎣 11. BAIT（餌管理）← NEW!
```
